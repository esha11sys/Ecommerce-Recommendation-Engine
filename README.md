# Ecommerce-Recommendation-Engine
This is my second python project based on the Recommendation models and Artificial intellegence with deep learning which gives the recommendations hybrid that is collaborative and content based filtering.
An AI-powered E-Commerce Recommendation Engine built using **Python** and **Flask**. This project recommends products to users by combining multiple recommendation techniques into a Hybrid Recommendation System.

---

##  Features

- Personalized Product Recommendations
- Hybrid Recommendation Engine
- Content-Based Filtering
- Collaborative Filtering
- Neural Collaborative Filtering
- Transformer-Based Recommendation
- A/B Testing
- Analytics Dashboard
- Clean and Responsive UI

---

## Recommendation Models Used

### Content-Based Filtering
Recommends products based on category and product similarity.

### Collaborative Filtering
Finds users with similar preferences and recommends products they liked.

### Neural Collaborative Filtering
Uses a neural-network-inspired approach for better recommendation quality.

### Transformer-Based Recommendation
Simulates sequential recommendation using transformer-inspired logic.

### Hybrid Recommendation
Combines all recommendation models to improve personalization and recommendation accuracy.
